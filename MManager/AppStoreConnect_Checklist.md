# App Store Connect 提出チェックリスト

以下の項目を App Store Connect で設定してください。

---

## 1. App 情報（App Information）

### コンテンツ配信権
- **このAppにはサードパーティのコンテンツが含まれていますか？**
  → 「いいえ」を選択
  → ✅ 「私はこのAppに対するコンテンツの権利を所有している、またはその使用許可を得ています」にチェック

### 主要カテゴリ
- **プライマリ**: ファイナンス（Finance）
- **セカンダリ**: ユーティリティ（Utilities）

### 年齢制限（Age Rating）
以下の質問にすべて「なし」と回答：

| 質問 | 回答 |
|------|------|
| まれ/軽度なアニメまたはファンタジーの暴力 | なし |
| まれ/軽度なリアルな暴力 | なし |
| まれ/軽度な性的表現やヌード | なし |
| まれ/軽度な成人向けテーマ | なし |
| まれ/軽度な言葉の表現または粗野なユーモア | なし |
| まれ/軽度な薬物またはアルコールの使用 | なし |
| まれ/軽度なホラーまたは恐怖に関するテーマ | なし |
| まれ/軽度な模擬ギャンブル | なし |
| 無制限のWebアクセス | いいえ |
| ギャンブルおよびコンテスト | いいえ |

→ 結果: **4+（全年齢対象）**

## 4. App のプライバシー

### プライバシーポリシー URL
```
（以下のいずれかにホスティングしてください）
```

**ホスティング方法（推奨順）：**

1. **GitHub Pages**（無料・簡単）
   - リポジトリの Settings → Pages → Deploy from branch
   - URL例: `https://username.github.io/MManager/privacy_policy.html`
   - `docs/privacy_policy.html` を push するだけ

2. **GitHub リポジトリ直接リンク**（暫定）
   - `https://raw.githubusercontent.com/username/MManager/main/docs/privacy_policy.html`

### App のプライバシー回答

**データ収集**: 「はい、このAppはデータを収集します」を選択

**収集するデータの種類:**

| データカテゴリ | データの種類 | 用途 |
|--------------|------------|------|
| 識別子 | デバイスID | 広告（サードパーティ広告） |

**トラッキング**: 「はい」を選択
- ✅ NSUserTrackingUsageDescription を使用しているため
- 目的: サードパーティ広告の配信

**データのリンク**: 「このデータはユーザーの ID にリンクされません」を選択
